# Player Quests

Progression track for the individual **player (user)** profile — distinct from
the cooperative track. See [`README.md`](../quests/README.md) for schema, status
legend, and the level-up rule.

> **Status: to be defined.**
> The cooperative quest structure (3 levels, required/optional schema) is
> established in [`coop.md`](./coop.md). Player-level quests will follow the
> same schema but track personal milestones (e.g. profile completion, learning,
> contributions) rather than legal-entity establishment.
>
> This document is intentionally a skeleton until the player progression
> design is finalized.
